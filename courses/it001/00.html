Work area

Your workspace

For the course, you have got two resources available, the resource website for your reading material and your Work area for your exercises.

Do all your exercises in the work area. For example, when an exercise mentions: "Create a new file", this means: "Create a new file in your work area."

Resource website

This is a small website with all the reading material for this course (You're on it at the moment). Here you will find all your reading material and your exercises.

Work area

This is your personal programming area where you can work on and submit your exercises. That area is a closed area, so you need an account there.

It's advisable to create a new folder for each exercise. This will keep your file order in ship shape! Ahoi!

Creating an account

In your browser, navigate to your Work area.
Click "Create a new account".
Enter your credentials and click "Sign up".
This creates your account and logs you into your work area.
To log out, click on the right hand user icon, then on "Sign out".
To log back in, click the left most login button and enter your credentials.

Creating a new project

Click on "File" -> "New" -> "Project" -> "Basic"
Enter the name of your new project
This will create a new project in your work area.

Creating a new Folder

Right click on your project, then click on "New" -> "Folder"
Enter the name of your new folder
This will create a new folder in your project.

Creating a new File

Right click on the parent folder, then click on "New" -> "File"
Enter the name of your new file
This will create a new file in the folder. The file is open in an editor on the right hand side of the window.
You can write your code into the editor.
Save your work by clicking on "File", then "Save".

Creating a test site

In your work environment you can create a test site where you can run your web pages and Javascript programs.

Click on "Sites"
This will navigate to the site manager.
Click "Create" to set up a new web site.
Enter the name of the website, then click "Submit".
This will set up a new web site.
The website won't work yet, because no folders have been mapped.
Click "Add", then the name of your project.
This will map your project and make it available on the website.
Click "Start" to activate the website.
This will start the website, making it available on the network. To see the website, navigate to the link indicated in blue.
This will navigate to your shiny new website.
When you are finished testing your website, please stop (deactivate) it again, so other users can test their websites. Sadly, only one website can run at any time.

Exercises

Familiarize yourself with your Work area.

  1. Create an account in your work area. Please let your tutor know the account details.
  2. In your work area, create a new project named it001.
  3. Create a new file test.html in project it001.
  4. Open the file test.html in your online editor. Paste the following code into the editor (save the file afterwards): 
    <html>
    <head>
        <script>window.alert ("Hello World");</script>
    </head>
    <body>
        Hello World in Javascript
    </body>
</html>
  5. In your work area, visit the site manager and create a new site site_it001.
  6. Add a Mapping to your project it001 in the site manager.
  7. Activate ("start") for your new website. You should be able to see it when you click the link provided.

Please stop your website again after starting it, otherwise other users can't start their web sites! Thank you!
