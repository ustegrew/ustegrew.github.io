Familiarize yourself with your work area.

  1. Create an account in your work area. Please let your tutor know the account details.
  2. In your work area, create a new project named it001.
  3. Create a new file test.html in project it001.
  4. Open the file test.html in your online editor. Paste the following code into the editor (save the file afterwards): 
                                    <html>
                                    <head>
                                        <script>window.alert ("Hello World");</script>
                                    </head>
                                    <body>
                                        Hello World in Javascript
                                    </body>
                                </html>
  5. In your work area, visit the site manager and create a new site site_it001.
  6. Add a Mapping to your project it001 in the site manager.
  7. Activate ("start") for your new website. You should be able to see it when you click the link provided.
Please stop your website again after starting it, otherwise other users can't start their web sites! Thank you!
