<static> TController.Notify(event)
For the given event, executes state transition. Transition is executed in several steps: 1. Worksheet executes pre transition actions (Save current exercise, collapse current solution etc.). 2. Worksheet sends "kSuccess" notification. 3. Now perform the transition. This also calls any worksheet specific handler.
