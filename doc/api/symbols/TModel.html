<private> <static> TModel._AssertIDUsable(id, testTypeUnique)
Tests whether the given id is aready stored, resp. not stored. The exact behaviour depends on the testTypeUnique parameter:
testTypeUnique hasID Result
ETestType.kSkip true OK
  false OK
ETestType.kIDMustExist true OK
  false throw exception
ETestType.kIDMustNotExist true throw exception
  false OK
Parameters:
{type} id
The UID to look for.
{type} testTypeUnique
A flag, determining the test specifics.
Throws:
{String}
Exception with message if an exercise with the same ID does (not) exist.
<private> <static> TModel._AssertIndexUsable(i)
Tests whether the given index i is within bounds. For i to be valid, the value of i must be in interval [0, n[ where n is the number of exercises stored.

The index to test.
