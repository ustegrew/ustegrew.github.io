<static> {int} TContext.Program_Step_GetState()
Returns the ID of the current step. This corresponds to the line number of the demo program's source code. Step ID's and line numbers always map as:
  • Step 0: Line 1
  • Step 1: Line 2
  • ...
  • Step n-1: Line n
Where n is the number of steps (hence, number of lines of source code) in the demo program.
Returns:
{int} The current step ID.
