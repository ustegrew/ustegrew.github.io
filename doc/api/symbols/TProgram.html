<private> <static> <constant> {JSObject} TProgram.kSchemaProgram
JS schema for a program description object. We use the courseware/util/validator/TValidatorJSON validator to perform the actual validation.
Note that global symbols have naming restrictions. A global symbol may only contain word characters or digits andlower or upper case characters or digits...
  • Must start with a word character [a-zA-Z],
  • followed by zero or more occurences of word characters or digits [a-zA-Z0-9].
Main reason is that the transition functions run in a custom environment which has some data members named with a starting underscore, and by using this naming convention we can avoid naming conflicts.
